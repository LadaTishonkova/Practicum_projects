# Practicum_projects
**Проект «Секреты Темнолесья»** 
[]
«Секреты Темнолесья» — это онлайн-игра, в которой каждый игрок следует по сюжету своего персонажа, выполняет задания и исследует огромный виртуальный мир.
Продажа «райских лепестков» — это основная часть дохода команды разработки. Команда игры планирует привлекать платящих игроков и продвигать покупку эпических предметов с помощью рекламы. 
*Цель проекта* — изучить влияние характеристик игроков и их игровых персонажей на покупку внутриигровой валюты «райские лепестки», а также оценить активность игроков при совершении внутриигровых покупок
